Max / MSP
오브젝트 사전 Vol.1

데이터 흐름 프로그래밍의 기본 단위들. 오브젝트를 클릭하면 인터랙티브 시뮬레이션이 실행됩니다.

디지털 프로세스 제작 과정은 단순 도구에 숙련의 관점이 아니라, 그 프로세스 과정에 사고방식에서 다른 생각의 사고방식을 가지는 즐거움이다.

Max에서 모든 것은 오브젝트(Object)와 연결(Patch Cord)로 이루어진다.
왼쪽 위의 인렛(inlet)으로 데이터가 들어오고, 아래 아웃렛(outlet)으로 나간다.
n = object 박스   m = message 박스

Section 05 · MSP Audio
오디오 오브젝트 (MSP)
~ 기호가 붙은 오브젝트는 오디오 신호를 처리. 샘플 레이트로 연속 계산.
Signal vs Message
~(틸드)가 붙으면 오디오 신호. 초당 44100개 샘플로 연속 처리. 일반 박스와 직접 연결 불가.
ezdac~ / adc~
ezdac~은 DAC(스피커 출력). 더블클릭으로 오디오 on/off. adc~는 마이크 입력.
파형 종류
cycle~(사인파) · phasor~(톱니파) · rect~(구형파) · tri~(삼각파) · noise~(백색소음) · saw~
볼륨 제어
*~ 0.2 로 신호에 0.2를 곱해 볼륨 감소. 신호값은 -1.0~1.0 범위.
∿
